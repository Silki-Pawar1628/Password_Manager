# 🔐 Password Manager Project

## 📌 Description
This is a simple Password Manager project developed using Python. It allows users to save and retrieve website passwords from a text file. The project also copies the retrieved password to the clipboard for quick use.

## 🚀 Features
- Save website and password
- Retrieve saved password
- Copy password to clipboard
- Simple command-line interface

## 🛠️ Technologies Used
- Python
- pyperclip
- File Handling

## 📂 Files
- main.py
- passwords.txt

## ▶️ How to Run
1. Install Python.
2. Install pyperclip:
   ```
   pip install pyperclip
   ```
3. Run:
   ```
   python main.py
   ```
